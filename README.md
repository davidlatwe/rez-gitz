
## rez-gitz

1. git clone from github
2. `rez-build` it


#### Install
```
$ git clone https://github.com/davidlatwe/rez-gitz.git
$ cd rez-gitz
$ rez build --install
```

#### Usage
```
$ rez-env gitz -- clone https://github.com/mottosso/rez-pipz.git --install
```
equivalent
```
$ rez-env gitz
> $ clone https://github.com/mottosso/rez-pipz.git --install
```
